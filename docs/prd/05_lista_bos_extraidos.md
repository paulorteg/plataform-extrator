# PRD 05: Lista de BOs Extraídos

## Objetivo

Criar uma tela empresarial de alto nível para triagem, gestão e acompanhamento das ocorrências extraídas.

## Informações na lista

1. Número do BO
2. Tipo de sinistro
3. Data do evento
4. Cidade e UF
5. Placa principal
6. CPF motorista mascarado
7. CNPJ vítima mascarado
8. Status
9. Confiança
10. Pendências obrigatórias
11. Responsável
12. Prioridade
13. Documento de origem

## UX

A tela deve ter dashboard de indicadores, busca global, filtros, Data Grid, badges de status, badges de confiança, ações contextuais e empty states.

## Segurança

Narrativa não aparece na lista. Dados sensíveis são mascarados. Ações respeitam permissões.

## Critérios de aceite

1. Lista paginada.
2. Filtros funcionam.
3. Busca funciona.
4. Dados sensíveis mascarados.
5. Ações sem permissão ocultas.
